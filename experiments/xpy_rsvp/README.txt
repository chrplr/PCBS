A Python script to display text, word by word, at the center of the screen.


   Requirements 

      pandas
      expyriment (see <http://expyriment.org>)

   Usage: 

     ./rsvp file.tsv

    where file.tsv is a TAB separeted values files with (at least) three colums: "word", "onset" and "duration" 
    (onset and duration are expressed in seconds)

   Note: You can interrupt the program by pressing 'Escape'.
