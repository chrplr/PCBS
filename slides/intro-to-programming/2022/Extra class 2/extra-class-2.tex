\begin{frame}{Coding Exercise}
\protect\hypertarget{coding-exercise}{}
\begin{itemize}
\tightlist
\item
  From \url{https://data.london.gov.uk/dataset/recorded_crime_summary}
  download the first file under Data \& resources named ``MPS Borough
  Level Crime (most recent 24 months)''. It contains the types and
  number of crimes per London borough between November 2020 and October
  2022.
\end{itemize}

Make sure to use functions whenever it's possible and avoid redundant
coding

\begin{itemize}
\item
  Get the total number of crimes across all boroughs in the oldest time
  period
\item
  Get all the months available in dataset
\item
  Get the index of a specific time period (month)
\item
  Get the total number of crimes across all boroughs for a specific
  month that you asked the user to input.
\item
  Get the evolution of one crime in one borough over all of the time
  periods.
\item
  Get the names of all boroughs in a appropriate data collection type.
\item
  Get the evolution of one crime over all boroughs over all the time
  periods
\end{itemize}
\end{frame}
